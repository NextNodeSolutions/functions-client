Add cn utility for className merging with clsx and tailwind-merge

This adds a powerful utility function for merging Tailwind CSS classes:
- Combines clsx for conditional class names
- Uses tailwind-merge to handle conflicting Tailwind classes
- Essential utility for UI components with dynamic styling
- Type-safe with full TypeScript support
- Compatible with all Tailwind CSS versions
